# ShareFiles
Приложение для обмена файлами в локальной сети (LAN) с поддержкой Drag & Drop.


Поддерживаются следующие языки: английский, русский, китайский (упрощённый, традиционный), испанский, португальский (бразильский), французский, немецкий, японский, арабский, итальянский, корейский, турецкий и польский.

## Настройка
Для отправки файлов необходимо запустить программу на двух компьютерах. На компьютере отправителя необходимо перенести нужные файлы или папку в интерфейс программы и выбрать компьютер с добавленным IP-адресом — после этого начнётся отправка файлов.<br><br>
Чтобы не выбирать IP-адрес можно запускать программу с параметром:<br>
>"C:\Program Files\ShareFiles\ShareFiles.exe" -d 192.168.0.2

Чтобы программа не запрашивала подтверждение при приёме файлов, добавьте IP-адрес в настройках.
